Cisco vManage will use built-in automation to do the following:

    Create WAN-VPC and spin up two Cisco Catalyst 8000V SD-WAN virtual routers
    Create site-to-cloud VPC
    Create site-to-site VPC
    Bring up Google Cloud Routers (GCR)
    Automate VPC peering and BGP routing between GCR and Cisco C8000V routers
